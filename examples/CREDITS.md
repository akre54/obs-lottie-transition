# Example Credits

The examples below are adapted for this plugin's slot and matte conventions and preserve credit back to the original LottieFiles works where applicable.

The `*-lf.json` files were derived from user-provided source animations and then modified for OBS transition testing with `[SlotA]`, `[SlotB]`, `[MatteA]`, and `[MatteB]` semantics where appropriate.

The older `*-inspired.json` variants were local reconstructions made before the original JSON files were available.
